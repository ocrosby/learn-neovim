## Reddit - r/neovim

**Title**: `[Resource] Learn Neovim - Complete Guide from Beginner to Advanced (v1.0.0)`

**Post**:

I've created a comprehensive learning resource for Neovim that takes you from absolute beginner to advanced user. Unlike distributions, this teaches you to build and understand your own config.

## What's Inside

**8 Progressive Modules** (15-25 hours total):
- Introduction, Basics, Intermediate
- Configuration (init.lua from scratch)
- Plugin Management (lazy.nvim)
- LSP setup and configuration
- Advanced Lua scripting
- Complete workflows (Git, debugging, testing)

**4 Example Configs** (all production-ready):
- **Minimal**: 31 lines, 0 plugins (learn fundamentals)
- **Basic**: ~300 lines, 14 plugins (general dev)
- **Python**: ~350 lines, 17 plugins (pytest, debugging, venv)
- **Web Dev**: ~340 lines, 18 plugins (JS/TS/React, ESLint, Tailwind)

Every line is explained!

**Plus**:
- 4 comprehensive cheatsheets
- 50+ hands-on exercises
- 100+ code examples
- Curated resources (articles, videos, communities)

## Quick Start

```bash
git clone https://github.com/ocrosby/learn-neovim.git
cd learn-neovim
nvim -u examples/basic/init.lua  # Try immediately
```

## Philosophy

- **Progressive learning**: No overwhelming config dumps
- **Understanding over copying**: Learn *why*, not just *what*
- **Build your own**: Create your perfect setup
- **Modern**: Lua, lazy.nvim, LSP (Neovim 0.9+)

## Who Is This For?

- Complete beginners (never used Vim)
- Vim users migrating to Neovim
- VSCode/JetBrains users curious about Neovim
- Anyone who wants to understand their editor

Hello Neovim community!

I'm excited to share **Learn Neovim** - a comprehensive educational resource I've built for learning Neovim from scratch.

## Background

After helping many beginners in various communities, I noticed they often:
- Copy configs without understanding them
- Struggle with the learning curve
- Don't know where to start
- Give up and go back to VSCode

Learn Neovim addresses all of these by providing a structured, progressive learning path.

## What's Included

**8 Complete Modules**:
1. Introduction & Setup
2. Basics (modes, navigation)
3. Intermediate (text objects, operators)
4. Configuration (init.lua from scratch)
5. Plugin Management (lazy.nvim)
6. LSP Setup & Configuration
7. Advanced (Lua scripting)
8. Workflows (Git, debugging, testing)

**4 Example Configurations**:
- Minimal (learning focused)
- Basic (general purpose)
- Python Development
- Web Development

All configs are production-ready and fully documented.

**Resources**:
- 4 comprehensive cheatsheets
- 50+ exercises
- Curated articles, videos, communities

## Philosophy

- **Progressive**: Start simple, build gradually
- **Understanding**: Learn why, not just what
- **Hands-on**: Exercises in every module
- **Modern**: Lua, lazy.nvim, LSP
