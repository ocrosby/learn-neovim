# Module 7: Development Workflows

Integrate Neovim into complete development workflows with Git, debugging, testing, and project-specific configurations.

## Learning Objectives

- Integrate Git workflows (fugitive, gitsigns)
- Set up debugging with nvim-dap
- Configure test runners
- Use project-specific configurations
- Build complete development environments

## Estimated Time

3-4 hours

## Prerequisites

- Completed [Module 6: Advanced](../06-advanced/README.md)
- Comfortable with your Neovim setup
- Understanding of development workflows

## Lessons

1. **[Git Integration](git-integration.md)**
2. **[Debugging](debugging.md)**
3. **[Testing](testing.md)**
4. **[Project-Specific Configuration](project-specific.md)**

## Why Workflows?

Go beyond editing to complete development: version control, debugging, testing, all within Neovim.

## Completion

After this module, you'll have a complete, personalized Neovim setup tailored to your development workflow.
